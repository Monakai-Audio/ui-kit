# Security Policy

## Supported Versions

We release security updates for the latest commit on the default branch of each
open-core repository. Older tags may not receive back-ports.

## Reporting a Vulnerability

If you discover a security vulnerability in any Monakai Audio open-core
repository, please report it responsibly:

* Email **security@monakaiaudio.com** with a detailed description and, if
  possible, steps to reproduce.
* Do not open a public issue for security-sensitive bugs.
* Allow reasonable time for assessment and remediation before public disclosure.

We will acknowledge receipt within 5 business days and keep you informed of our
progress.

## Scope

This policy covers the open-core repositories under the
[monakai-audio](https://github.com/monakai-audio) GitHub organization. It does
not cover proprietary products, plug-ins, or premium content, which are handled
through separate channels.
